### 初始化项目

1. 初始化项目:npx create-react-app hkzf-mobile
2. 启动项目,在项目根目录执行命令:yarn start
3. 调整项目中 src 目录结构,如下:

```
src/
  assets/  #资源(图片、字体图标等)
  components/  #公共组件
  pages/  #页面
  utils/  #工具
  App.js  #跟组件(配置路由信息)
  index.css  #全局样式
  index.js  #项目入口文件(渲染跟组件、导入组件库等)
```
